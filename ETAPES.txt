ETAPES du programme

1ère ETAPE
Chargement le fichier INI
	=> Resolution
	=> Volume (Son/Musique(s))

Charger les fichiers de propriétés
	=> Configuration:
		- resources.def
			ex: (FORET: 90% arbre, 10% eau)
		- diets.det
				-> Nom du régime
				-> Liste des ressources tolérées dans l'alimentation

		- animalsAlreadyExisting.def
				-> Régime alimentaire
				-> Taille
				-> Poids
				-> Charset
		- proceduralAnimals.def
			ex: CARNIVORE doit être plus rapide que les HERBIVORES...
				-> Poids min/max
				-> Taille min/max
				-> Charset (image animal)
				-> Régime
				-> Température d'acclimatation
				-> Espérance de vie (exprimé en jours)
		- weathers.def
				-> Nombre de météo existantes
				-> Nom de chaque météo
		- meteo1.def/meteo2.def...
				-> Nom
				-> Quantité d'eau
				-> Changement température
				-> Ensoleillement
		- climates.def
				-> Nom
				-> Température
				-> Météo par défaut
		- biomes.def
				-> Nom
				-> Tileset
				-> Climat
				-> Propriétés du sol
		- definition.def:
				-> Fréquence à laquelle l'animal doit se nourrir suivant ses besoins
				-> Fréquence des catastrophes (0 aucune, sinon entre 1 et 1000000~)
